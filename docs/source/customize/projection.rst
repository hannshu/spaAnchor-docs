Customize :meth:`~spaAnchor.HANN.projection`
---------------------------------------------

The projection module is responsible for generating the cell-level 
and niche-level latent features, which are then used in subsequent 
HNN pair identification and feature translation.

By default, the :meth:`~spaAnchor.HANN.projection` function uses PCA
or Harmony to generate the cell-level latent features from the 
preprocessed input features. The niche-level latent features are then 
generated by aggregating the cell-level latent features based on the spatial graph.
We find that the default projection method works well in most cases, but users can
customize the projection step by overriding the :meth:`~spaAnchor.HANN.projection` 
function according to their specific needs.

Such as if the default projection method performs poorly on a given dataset, for example, 
due to batch effects that cannot be adequately corrected by Harmony, users may 
change to an alternative dimensionality reduction method, such as scVI, to generate 
the cell-level latent features by overlaid :meth:`~spaAnchor.HANN.projection_cell` function.


.. code:: python

    import spaAnchor as sa

    class custom_scvi(sa.HANN):

        def __init__(self, **kwargs):
            super().__init__(**kwargs)

        def projection_cell(
            self,
            adata_list: List[sc.AnnData],
            **kwargs
        ):
            import scvi
            scvi.settings.seed = 0

            adata = sc.concat(adata_list, label="batch", keys=["0", "1"])
            adata.layers["counts"] = adata.obsm["X_raw"]
            scvi.model.SCVI.setup_anndata(
                adata,
                layer="counts",
                batch_key="batch"
            )
            model = scvi.model.SCVI(adata)
            model.train(early_stopping=True, enable_progress_bar=True)
            adata.obsm['cell_latent'] = model.get_latent_representation(adata).astype(np.float32)

            return [adata[id == adata.obs["batch"]] for id in ["0", "1"]]    


Moreover, users can also customize the niche-level latent feature generation or completely
change both the cell-level and niche-level latent feature generation with more complex
ways by overriding the :meth:`~spaAnchor.HANN.projection` function.
